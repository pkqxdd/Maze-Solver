Friday, October 13¶

I finished writing the Project Proposal along with a bunch of research on what type of sensor I should be using. The Reference Sources in my project list are very helpful, of course, but none of them appears to be able to handle complicated mazes like the one with loops, or circles. So I decide to make one with such ability. I know this will be a hard task, but I’m ready. It’s time to put the graph search algorithms I learned into real use.
